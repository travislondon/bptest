205. Simple Association

Simple Association(Rel_ID)

Identifier *: Rel_ID

Identifier *2:

Identifier *3:

Description: A simple association (R_SIMP) is an association between two classes which are formalized with referential attributes in one of the classes. The class that holds the referential attributes for the association is know as the referring class in the association (R_RGO) and is the formalizer of the association (R_FORM). The class that participates in the association and carries the identifying attributes used to formalize the association is the referred to class (R_RTO) and is a participant in the association (R_PART).

Simple Association.Rel_ID

Attribute Type: Referential Attribute

Refers to: Association.Rel_ID (R206)

208. Linked Association

Linked Association(Rel_ID)

Identifier *: Rel_ID

Identifier *2:

Identifier *3:

Description: A linked association (R_ASSOC) is an association between two classes ( R_AONE and R_AOTH) which is formalized in an associative class (R_ASSR). The class that holds the referential attributes for the association is known as the referring class in the association (R_RGO) and the class that participates in the association and carries the identifying attributes used to formalize the association is the referred to class (R_RTO). Linked associations require formalization of the association in a class called the associator (R_ASSR).

Linked Association.Rel_ID
